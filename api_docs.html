API Documentation

Scraping API

This API is used to scrape websites.

Request Example:

GET http://localhost:8000/api/scrape?website_url=https://nextjs.org/docs

Response:

{ 'msg': 'Success in scraping website' }

Chatbot API

This API is used for chatting with a scraping chatbot.

Request Example:

GET http://localhost:8000/api/chat?user_message=hello

Response:

"Hello! How can I assist you today?"
